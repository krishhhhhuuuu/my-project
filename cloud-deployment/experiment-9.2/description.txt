## Description

This experiment demonstrates a GitHub Actions workflow that automates building and testing of a simple Node.js project.
Whenever a new commit is pushed to the repository, the pipeline automatically runs the following steps:

1. Checkout Repository ‚Äì Clones the latest code from GitHub.
2. Setup Node.js ‚Äì Installs Node.js in the workflow runner.
3. Install Dependencies ‚Äì Installs project dependencies using npm.
4. Run Build ‚Äì Executes the build command defined in `package.json`.
5. Run Tests ‚Äì Executes the test script.

### 3. .github/workflows/main.yml

```yaml
name: CI/CD Pipeline

on:
  push:
    branches:
      - main

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
      - name: Checkout repository
        uses: actions/checkout@v3

      - name: Setup Node.js
        uses: actions/setup-node@v3
        with:
          node-version: '18'

      - name: Install dependencies
        run: npm install

      - name: Build project
        run: npm run build

      - name: Run tests
        run: npm test
```

1. Push Code to GitHub:

2. Workflow Execution:
